# constant FSPath

Creates path to filesystem object. Can be chained with any key name wich act as path segment.

View Source index.ts, line 44

Example
FSPath(__dirname).node_modules //work as path.join(__dirname, "node_modules")
FSPath(__dirname)["package.json"] //work as path.join(__dirname, "package.json")
// When path completed you can get further operation to call it as function.
FSPath(__dirname).node_modules().asDir()
FSPath(__dirname)["package.json"]().asFile()
//At this point you can use FSDir

# constant mkdtemp

Creates a unique temporary directory. Generates six random characters to be appended behind a required prefix to create a unique temporary directory. Due to platform inconsistencies, avoid trailing X characters in prefix. Some platforms, notably the BSDs, can return more than six random characters, and replace trailing X characters in prefix with random characters. The created directory path is passed as a string to the callback's second parameter.
